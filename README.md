# EMI Calculator

A simple Python script to calculate monthly EMI (Equated Monthly Installment) based on loan amount, interest rate, and loan duration.

## 📥 Input:
- Principal amount
- Annual interest rate (in %)
- Loan duration (in years)

## 📤 Output:
- Monthly EMI
- Total payment
- Total interest

## 🛠️ How to Run:
```bash
python EMi calculator.py
